Array Concatenation

Description :

Javascript offers a handy concat method for array concatenation. The concat method can concatenate arrays into a single array. The method takes as many arguments as needed and returns a new concatenated array. The arrays sent as arguments or the pre-existing arrays aren't modified by this method.

Syntax : let changed = original.concat(original_array, original_second_array, ...);
